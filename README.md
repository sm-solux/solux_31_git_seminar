# solux_31_git_seminar
솔룩스 31기 깃허브 세미나 레포지토리입니다.

1. 레포지토리를 clone 합니다.
2. feature/본인깃허브아이디 형식의 branch를 생성합니다. ex) feature/solux
3. "자기소개.md" 파일을 변경합니다. Markdown 문법을 이용해 간단한 자기소개를 남겨주세요.
4. 파일명을 "이름_자기소개.md" 로 변경합니다.
5. main 브랜치로 Pull Request를 보내주세요. PR 제목은 "이름_자기소개" 입니다.
6. 다른 부원의 자기소개 PR을 찾아가서 최소 2개 이상의 리뷰를 달아주세요.
